# AI流程编排工具

基于 LogicFlow 构建的模仿腾讯元宝风格的 LLM 流程编排界面，支持拖拽式创建 AI 工作流。

## 🎯 项目特点

- **🎨 现代化界面设计** - 参考腾讯元宝的设计风格，提供优雅的用户体验
- **🔧 丰富的节点类型** - 支持开始、结束、大模型、知识库、条件判断、API调用等多种节点
- **🖱️ 拖拽式操作** - 直观的拖拽界面，轻松构建复杂工作流
- **⚙️ 详细配置选项** - 每个节点都有专门的配置面板，支持参数定制
- **💾 数据持久化** - 支持工作流的保存和加载，数据不丢失
- **📊 实时日志** - 运行过程可视化，便于调试和监控
- **📱 响应式设计** - 适配不同屏幕尺寸，移动端友好

## 🚀 快速开始

### 1. 直接使用

将所有文件下载到本地，然后在浏览器中打开 `index.html` 即可开始使用。

### 2. 基本操作

1. **添加节点**: 从左侧节点库拖拽节点到画布
2. **连接节点**: 点击节点的锚点并拖拽到目标节点
3. **配置节点**: 点击节点后在右侧面板进行配置
4. **运行工作流**: 点击顶部工具栏的"运行"按钮
5. **保存/加载**: 支持将工作流保存到本地存储

## 🔧 支持的节点类型

### 基础节点
- **开始节点** - 工作流的起始点
- **结束节点** - 工作流的终止点

### AI节点
- **大模型节点** - 配置各种 LLM 模型（GPT、Claude、通义千问等）
- **知识库节点** - 集成知识库搜索功能
- **提示词节点** - 专门的提示词处理节点

### 逻辑节点
- **条件判断节点** - 基于条件的分支控制
- **循环节点** - 循环执行逻辑
- **分支节点** - 多路分支控制

### 插件节点
- **API调用节点** - 调用外部 API 接口
- **数据库节点** - 数据库操作
- **文件处理节点** - 文件读写处理

## 📋 详细功能

### 节点配置
每种节点类型都有专门的配置界面：

#### 大模型节点配置
- 模型选择（GPT-3.5、GPT-4、Claude-3、通义千问）
- 温度参数调节
- 最大Token数设置
- 系统提示词和用户提示词

#### 知识库节点配置
- 知识库选择
- 搜索类型（语义搜索、关键词搜索、混合搜索）
- 返回条数和相似度阈值

#### API调用节点配置
- API地址和请求方法
- 请求头和请求体配置
- 支持GET、POST、PUT、DELETE等方法

### 界面功能
- **顶部工具栏** - 运行、保存、清空等快捷操作
- **左侧节点库** - 按类别组织的节点拖拽面板
- **中央画布** - 主要的流程编辑区域
- **右侧属性面板** - 节点配置和属性编辑
- **底部日志面板** - 运行日志和状态显示
- **画布控制** - 缩放、适应视图等操作
